# myContactList
An simple Contact list web application using the MEAN Stack (Mongo, Express, Angular, Node).

Requirements

1. Nodejs
2. Mongodb

How to install?

On your terminal, type these commands in order

1. git clone https://github.com/mcariel/myContactList
2. cd myContactList
3. npm install
4. bower install
5. to use, type 'gulp'
